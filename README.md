# VITS iResource Hub

[![Live Site](https://img.shields.io/badge/Live-Website-brightgreen?style=for-the-badge)](https://manideepvelchuri.github.io/vits-iresource-hub/)

A centralized academic resource directory designed specifically for students of the **Vignan Institute of Technology and Science (VITS)**. This portal brings together JNTUH syllabus files, course books, lab guides, core engineering software downloads, placement prep links, and official college portals into a fast, unified, mobile-responsive dashboard.

## 🚀 Live Link
Access the hub here: **[https://manideepvelchuri.github.io/vits-iresource-hub/](https://manideepvelchuri.github.io/vits-iresource-hub/)**

---

## 🌟 Key Features

* **📚 Unified Academic Library**: Semester-wise accordions mapping JNTUH syllabus structures directly to Google Drive folders containing standard reference textbooks, handwritten lecture notes, and lab manuals.
* **💿 Core Software Download Center**: Categorized links to official downloads and free/open-source alternatives for 35+ core tools including MATLAB, Vivado, Keil, LTspice, Proteus, and AutoCAD.
* **🎯 GATE & Placements Portal**: Curated lists of recommended books, exam guides, mock portals, and coding preparation platforms (LeetCode, HackerRank, GeeksforGeeks) for technical recruitment.
* **🏫 College Portals Integration**: Direct dashboard links to the **VITS Attendance Portal** (`http://103.52.36.11/Attendance/index.html`) and the **VITS Official Website** (`https://vignanits.ac.in/`).
* **⚡ Optimized Page Speed**: Light-weight static pages (HTML, Vanilla CSS, JS) optimized with deferred script loading and image lazy loading to achieve maximum Google PageSpeed and Lighthouse scores.

---

## 🛠️ Tech Stack & SEO Configurations

* **Frontend**: Semantic HTML5, CSS3 (Modern dark/light theme utilizing HSL color systems), and Vanilla JavaScript.
* **SEO & Crawl Optimization**:
  * Unified sitemap (`sitemap.xml`) and search-engine index guidelines (`robots.txt`).
  * Unique page-level `<link rel="canonical">` and description meta tags.
  * Schema.org Structured JSON-LD metadata for rich Google Search indexing.
  * Complete Open Graph metadata on all pages for rich messaging link previews.
